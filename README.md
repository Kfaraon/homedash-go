# HomeDash – мониторинг домашних сервисов на Go

[![Go Version](https://img.shields.io/badge/go-1.21+-blue.svg)](https://golang.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**HomeDash** – это лёгкий дашборд для отслеживания доступности ваших домашних сервисов (Proxmox, AdGuard, Home Assistant, роутер и любых других).  
Проект написан на Go и не требует сложной настройки: вы просто описываете сервисы в JSON-конфиге, а веб-интерфейс показывает их статус в реальном времени.

## ✨ Возможности

- **Группировка сервисов** – объединяйте сервисы в логические группы (например, «Виртуализация», «Сеть», «Умный дом»).
- **Двойная проверка** – HTTP/HTTPS и ICMP (ping) для каждого сервиса (можно использовать один из методов или оба).
- **Асинхронное обновление** – статусы проверяются параллельно, данные кэшируются на 3 секунды.
- **Адаптивный интерфейс** – корректно отображается на ПК, планшетах и смартфонах.
- **Тёмная / светлая тема** – переключается с сохранением выбора в `localStorage`.
- **Подсказки при наведении** – на иконках HTTP и ICMP отображается URL или IP сервиса.
- **Минималистичный дизайн** – вдохновлён GitHub и Prometheus.

## 📦 Требования

- Go 1.21+ (для локальной сборки)
- Docker (опционально, для запуска в контейнере)
